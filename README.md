# 🚀 EKS Cluster Setup using Terraform

## 📌 Project Overview
This project demonstrates how to provision an AWS EKS cluster using Terraform and deploy a containerized application using Kubernetes.

## 🛠 Tech Stack
- AWS (EKS, VPC, EC2)
- Terraform
- Kubernetes
- Docker (Nginx)

## ⚙️ Features
- Custom VPC with public subnets
- EKS cluster with managed node group
- Kubernetes Deployment & LoadBalancer Service
- Public access via AWS ELB

## 🚀 Steps to Run
1. terraform init
2. terraform apply
3. Configure kubectl
4. Deploy Kubernetes manifests

## 🌐 Output
Application exposed via AWS LoadBalancer
